About the Course

Web development refers to the process of creating and maintaining websites and web applications that are accessible via the internet. It involves a combination of various technologies, programming languages, frameworks, and tools to build the structure, design, functionality, and interactivity of websites.

Course Objectives:

  • To inculcate understanding of World Wide Web, Internet, the concepts of web applications development and web programming languages.
  • To introduce selection of appropriate concepts of internet and web programming such as HTML, CSS, JavaScript, and other server-side scripting languages.
  • To infuse skills of combining different components and technologies to design a web application for real world problem
  • Test the components of WWW, HTML tags, CSS properties, client-side and server-side programming concepts, web data representation formats and AJAX components using different web development tools.

Target Audience:

  • Beginners & Enthusiasts: Individuals with little to no coding experience who want to learn the basics of building websites using HTML, CSS, and JavaScript.
  • Career Switchers: Professionals from other fields looking to transition into web development, seeking a comprehensive course covering both front-end and back-end technologies.
  • Designers & Freelancers: Graphic designers and freelancers aiming to expand their skill set by acquiring front-end development abilities to create interactive web interfaces.

Course Outline

  • Module 1 : Introduction to World Wide Web
  • Module 2 : Markup Languages and Building Web Pages
  • Module 3 : Style sheet Languages and Presentation of Web Pages
  • Module 4 : Client-side Programming
  • Module 5 : Server-side Programming
  • Module 6 : Representation of Web Data, AJAX and Web Security
